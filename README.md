# stache #

*Stache* is a rendering engine heavily inspired by [mustache].

[mustache]: http://mustache.github.com/

## Installation ##

To use it in your own project, add the following lines to your `composer.json`.

    "require" : {
        "rtens/stache" : "*"
    },
    "minimum-stability": "dev"
